# TCG Companion

紙のカードゲーム（アナログTCG）を遊ぶときの、スマホ用の対戦支援ツールです。
**ライフカウンター**と**対戦記録**がひとつになっていて、コイントスやダイスの結果、マリガン数が
そのまま戦績に残ります。

**▶ 使ってみる: https://blueberryiscream-ops.github.io/tcg-companion/**

インストール不要・完全無料・広告なし・オフラインでも動きます。

---

## できること

- **ライフカウンター** — 1対1でも多人数（統率者戦など）でも。長押しで一気に増減、Undo付き
- **コイントス / ダイス** — d4・d6・d8・d10・d12・d20・d100。履歴も残る
- **先攻後攻の決定** — ダイスやコインで決めて、そのまま記録に残る
- **マリガン数のカウント** — これも記録に残る
- **対戦の記録と分析** — 対面デッキ別・先手後手別・マリガン別の勝率、勝率の推移グラフ
- **CSV書き出し** — 表計算ソフトで自由に分析できる
- **ゲームごとのカスタマイズ** — 開始ライフ、使うダイス、独自カウンター、テーマ色、壁紙
- **バックアップ** — データをファイルに書き出して、別の端末に読み込める

## はじめかた（3ステップ）

1. スマホのブラウザで **[アプリを開く](https://blueberryiscream-ops.github.io/tcg-companion/)**
2. **ホーム画面に追加する**
   - iPhone: Safariで開き、下の「共有」ボタン → 「ホーム画面に追加」
   - Android: Chromeで開き、右上の「⋮」→ 「ホーム画面に追加」
3. ホーム画面のアイコンから開いて、ゲームを選ぶ

> **ホーム画面への追加は必ず行ってください。**
> iPhoneはブラウザのタブで開いたままだと、しばらく使わない間に
> アプリの保存データ（対戦記録）を自動で削除することがあります。
> ホーム画面に追加すると、その対象から外れます。

くわしい使い方は、アプリ内の **設定 → 使い方・よくある質問 → はじめての方へ** にあります。

## 技術仕様 / プライバシー

「知らないURLを開くのは気が引ける」という方へ。中身はこうなっています。
すべてこのリポジトリのソースで確認できます。

### サーバーがありません

完全なクライアントサイドアプリです。バックエンドもAPIも、データベースサーバーもありません。
GitHub Pages から静的ファイルを配信しているだけです。

| 項目 | 実際のところ |
|---|---|
| アカウント登録 | **不要**（そもそも仕組みがない） |
| データの保存先 | **あなたの端末の中だけ**（IndexedDB / localStorage） |
| サーバーへの送信 | **一切なし**。ソースに `fetch` / `XMLHttpRequest` / `WebSocket` が存在しません |
| アクセス解析・トラッキング | **なし**（Google Analytics等の類は一切入っていません） |
| 広告 | **なし** |
| Cookie | **使用しません** |
| 外部CDN・外部フォント | **なし**。すべてバンドルに同梱、実行時に外部へ取りに行きません |
| 通信が発生する場面 | **アプリ本体を最初にダウンロードする時だけ**。以降はオフラインで動作します |
| 費用 | **無料**。運用にランニングコストがかからない設計です |

### 使用する端末機能（すべて任意・拒否できます）

| 機能 | 用途 | 既定 |
|---|---|---|
| ストレージの永続化要求 | 端末がデータを勝手に消さないようにする | 起動時に要求（拒否されても動作します） |
| 画面オフ防止 (Screen Wake Lock) | 対戦中に画面が消えないようにする | **オフ** |
| 全画面表示 (Fullscreen API) | ブラウザのバーを隠す | **オフ** |
| 画像の読み込み | 背景の壁紙に使う | 任意。**選んだ画像は端末内に保存され、送信されません** |
| ファイルの保存 | CSV・バックアップの書き出し | あなたがボタンを押した時だけ |

カメラ・マイク・位置情報・連絡先・通知は**一切使いません**。

### データを消したいとき

ホーム画面からアイコンを削除し、ブラウザの設定でこのサイトのデータを消せば完全に消えます。
消す前に「設定 → データのバックアップ」からファイルに書き出しておけば、後で復元できます。

### 構成

- TypeScript / React / Vite / Tailwind CSS
- Dexie（IndexedDBのラッパー）でローカル保存
- vite-plugin-pwa（オフライン動作・ホーム画面への追加）
- 実行時の依存: `react` `react-dom` `dexie` `dexie-react-hooks` `@dnd-kit/*` `lucide-react`

### 動作環境

- **Android / Chrome**: 検証済み
- **デスクトップ (Chrome / Edge / Firefox)**: 動作します
- **iPhone / iPad**: 動作する想定ですが**実機での検証中**です。
  なお iOS では「画面オフ防止」「全画面表示」は使えません（Appleが未対応のため、
  設定項目自体が表示されません）。
