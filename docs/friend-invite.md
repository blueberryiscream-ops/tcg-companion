# 友人に送る案内文（LINE等にそのまま貼れる）

> LINEは書式装飾が効かないため、記号と改行だけで組んである。URL以外は改変自由。
> iPhone / Android で最初の一手が違うので、相手に合わせて該当ブロックだけ残してもよい。

## iPhone の友人に送る版（＝最初のテスター用）

```
例のカードゲーム用ツール、形になったから使ってみて！
インストール不要、無料、広告なし。オフラインでも動くよ。

https://blueberryiscream-ops.github.io/tcg-companion/

⚠️ 最初にこれだけやってほしい（重要）
1. 上のリンクを Safari で開く
2. 下の「共有」ボタン（□に↑のマーク）をタップ
3. 「ホーム画面に追加」を選ぶ
4. 以後はホーム画面のアイコンから開く

これをやらないと、iPhoneが数日〜数週間でアプリの中の記録を
勝手に消してしまうことがある（iOSの仕様）。
追加すれば消えなくなるので、必ず最初にお願い！

📖 使い方
アプリの「設定」→「使い方・よくある質問」→「はじめての方へ」に
一通り書いてある。最初にざっと見てもらえると早いと思う。

🔄 アップデートが反映されない時
アプリを一度「完全に」閉じて（アプリ切替画面から上にスワイプ）、
開き直すと最新版になる。

🙏 お願い
まだテスト段階なので、
・動かない
・見た目が崩れる
・ここが使いにくい
があったら遠慮なく教えて。スクショがあると助かる！
```
