# テスター用チェックリスト（iPhone / 最初の友人に渡す用）

本ツールは Android(Chrome) と WebKit自動テストでしか検証できていない。
**iPhone実機でしか分からないこと**に絞ってある。全部やらなくていい。上から順に大事。

本番URL: https://blueberryiscream-ops.github.io/tcg-companion/

---

## A. インストール（ここが一番大事）

- [ ] **A-1** Safariでリンクを開いた時、「ホーム画面に追加してください」というオレンジ色の案内が出るか？
- [ ] **A-2** 「共有」→「ホーム画面に追加」ができるか？アイコンはちゃんと表示されるか？
- [ ] **A-3** ホーム画面のアイコンから開いた時、**オレンジの案内が消えている**か？
- [ ] **A-4** アイコンから開いた時、**Safariのアドレスバーや下部バーが出ていない**か？（アプリっぽい見た目になっているか）

## B. データが消えないか（最重要・時間がかかる）

- [ ] **B-1** 対戦を1件記録する → アプリを完全に閉じる → 開き直して**記録が残っている**か？
- [ ] **B-2** **数日〜1週間ほど空けて**もう一度開く → 記録がまだ残っているか？
      （↑これが今回一番確かめたいこと。iPhoneが勝手にデータを消さないかの確認）
- [ ] **B-3** 「設定」→ バックアップの書き出しを試す。**ファイルとして保存できる**か？
      （iPhoneはここが不安定な可能性あり。保存できない／何も起きないなら要報告）
- [ ] **B-4** 記録画面の CSV書き出しも同様に保存できるか？

## C. 基本の使い心地

- [ ] **C-1** ライフの ± ボタン、長押しでの連続増減、Undo（戻す）が効くか
- [ ] **C-2** コイントス・ダイス・先攻後攻を決める流れが自然か
- [ ] **C-3** マリガンのカウントができるか
- [ ] **C-4** 対面プレイ（相手側が180度回転する表示）で、**相手側の数字が読めるか・押し間違えないか**
- [ ] **C-5** 対戦を記録 → 履歴と勝率が正しく出るか
- [ ] **C-6** ゲームを追加／編集／削除ができるか。削除の警告に「記録◯件も消えます」と件数が出るか

## D. 見た目

- [ ] **D-1** 壁紙（背景画像）を自分の写真から設定できるか。設定しても**数字や文字が読めるか**
- [ ] **D-2** ノッチ（画面上部の切り欠き）や縦長画面でレイアウトが崩れていないか
- [ ] **D-3** 暗い所・明るい所で見づらくないか

---

## 報告不要（既知の仕様・こちらで把握済み）

- 「画面オフ防止」「全画面表示」のトグルが**表示されない**、または効かない
  → iOSが対応していないため。仕様です。
- 対戦の**最中に**そのゲームの「開始ライフ」を編集すると、盤面がリセットされる
  → 既知の制限。対戦前に設定してもらう想定。

## 報告してほしい形

```
・何をしたか（例: 設定→バックアップの書き出しを押した）
・何が起きたか（例: 何も起きない）
・どうなると思っていたか（例: ファイルが保存されると思った）
・できればスクリーンショット
```

「使いにくい」「なんかダサい」みたいな**感覚の指摘も歓迎**。直せます。
